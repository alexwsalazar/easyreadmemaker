# Easy Readme Maker

A easy application that promps you to enter information and makes a business readme document


## Run Locally

Clone the project

```bash
  git clone https://github.com/alexwsalazar/easyreadmemaker
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
